Modelo de fogo do DOOM de PS1

Esse Ã© um pequeno experimento de como criar o fogo do DOOM de PS1 com java script puro, com a finalidade de treinar um pouco de JS, HTML e CSS. Foi um dos primeiros projetos que eu me propus a fazer no inicio dos meus estudos. Criando uma tabela de 70 x 40 e definindo cada celula com um valor de 0 a 36, onde o 36 corresponde a uma cor clara de fogo mais intenso e 0 de cor escura sem fogo algum. As celulas de cima sempre tem um valor menor que as celulas de baixo, fazendo com que a medida que as celulas de baixo variem sua intencidade, um efeito cascata segue para os pixels mais altos criando o efeito do fogo.
